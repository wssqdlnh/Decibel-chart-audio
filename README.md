# Decibel-chart-audio
Instructions for use:
After selecting the audio file, the system will analyze the entire audio file to generate an accurate decibel chart
Click on the decibel chart to jump to the corresponding time position
Dragging on the decibel chart allows you to preview and select the precise time and location
When playing, the decibel chart will display the waveform of several seconds before and after the current playback time as the center
Dragging the progress bar can also jump to the playback position, and the decibel chart will be updated synchronously
使用说明:
选择音频文件后，系统会分析整个音频文件生成精确的分贝图
点击分贝图可以跳转到对应时间位置
在分贝图上拖动可以预览并选择精确的时间位置
播放时，分贝图会以当前播放时间为中心显示前后数秒的波形
拖动进度条也可以跳转播放位置，分贝图会同步更新
